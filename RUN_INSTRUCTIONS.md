# How to Run the Forest Plot Generator

Since you have the code on your computer, follow these simple steps to start the application:

1.  **Open the Terminal**:
    *   If you are in VS Code, look at the top menu, click **Terminal**, then **New Terminal**.

2.  **Type the Start Command**:
    *   Click inside the terminal window that appeared at the bottom.
    *   Type the following command exactly:
        ```bash
        npm start
        ```

3.  **Press Enter**:
    *   Hit the `Enter` key on your keyboard.

The application window should open up shortly.

## Troubleshooting
If you see any red text or errors:
*   Ensure you are in the correct folder (`forest-plot-app`).
*   Try closing the terminal and opening a new one.
